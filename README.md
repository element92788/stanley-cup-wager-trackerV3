# Stanley Cup Wager Tracker

Clean full project.

## Rules

- Brett = Vegas Golden Knights
- Dad = Carolina Hurricanes
- $10 per Stanley Cup Final game won
- $100 bonus to Cup winner
- Regular season games appear in history only

## Official Finals schedule

Uses the NHL.com schedule:
Game 1 June 2, Game 2 June 4, Game 3 June 6, Game 4 June 9, Game 5 June 11, Game 6 June 14, Game 7 June 17, all at 8 PM ET.

## Deploy

```bash
npm install
npm run build
```

Push to GitHub, then Vercel will deploy.
