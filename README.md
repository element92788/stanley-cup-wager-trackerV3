# Stanley Cup Wager Tracker

Mobile-first Next.js app for Brett vs Dad's Stanley Cup wager.

## Wager

- Brett = Vegas Golden Knights
- Dad = Carolina Hurricanes
- $10 per game won
- $100 bonus to the Stanley Cup winner

## Features

- Countdown to next puck drop
- Live score panel
- Automatic NHL public API polling
- Series tracking
- Wager math from Brett or Dad's side
- Mobile/PWA ready
- Vegas gold + Hurricanes red theme

## Deploy to Vercel

1. Upload the extracted project files to GitHub.
2. Make sure the GitHub repo root contains:
   - `app/`
   - `components/`
   - `lib/`
   - `public/`
   - `package.json`
3. Import the repo into Vercel.
4. Framework Preset should be `Next.js`.
5. Click Deploy.

Do not upload the ZIP itself to GitHub. Upload the extracted contents.
